# FlappyGradient

A Web3 Play-to-Earn Flappy Bird clone built on OpenGradient (Base Sepolia).

## Features
- **Play-to-Earn:** Deposit $OPG tokens to play.
- **Web3 Integration:** Connects to Base Sepolia testnet.
- **Modern UI:** Full-screen canvas, neon aesthetics, particle effects.
- **Physics:** Smooth flight mechanics.

## How to Play
1. Open the game.
2. Send 0.001 $OPG to the generated Game Wallet.
3. Click "Start Game".
4. Tap/Click to fly.

## Deployment
This project is ready for deployment on Vercel. Simply import this repository and deploy.
